# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our pledge

We aim to keep project discussions respectful, focused, and useful. Disagreement
is fine; harassment, abuse, and personal attacks are not.

## Scope

This applies to GitHub issues, pull requests, discussions, release channels, and
other project spaces.

## Enforcement

Maintainers may remove comments, close threads, or block participants who make
the project hostile or unsafe for others.
